Задание 2
Создайте класс, который позволит выполнять мониторинг ресурсов, используемых программой.
Используйте его в целях наблюдения за работой программы, а именно: пользователь может
указать приемлемые уровни потребления ресурсов (памяти), а методы класса позволят выдать
предупреждение, когда количество реально используемых ресурсов приблизиться к
максимально допустимому уровню.

Решение:
-1. Создать класс MemoryMonitor который содержит 2 метода: bool IsLimitExceeded, void WarningMsg;
-2. В методе IsLimitExceeded сделать проверку, если заданный размер пользователем не превышает размер возвращаемого значения метода GC. GetTotalMemory, возвращаем true;
-3. Создать класс MyArray, в теле класса создать массив на 100 000 000 элементов и метод, который будет выводить информацию о заполнении массива;
-4. В Program создать экземпляр класса Timer(), в конструктор передать экземпляр класса MemoryMonitor, вызвать метод IsLimitExceeded, вторым параметром передать обьекту-строку, третьим 0 - запускаем мониторинг сразу после создания экземпляра, и четвертым 200 милисекунд - частота мониторинга.
-5. Создать массив типа MyArray размерностью в 15000 элементов и заполнить массив в цикле вызвав при этом метод, который выведет информацию о заполнении;
