Задание 3
Создайте программу, в которой предоставьте пользователю доступ к сборке из Задания 2.
Реализуйте использование метода конвертации значения температуры из шкалы Цельсия в
шкалу Фаренгейта. Выполняя задание используйте только рефлексию.

Решение:
-1. Скопировать библиотеку ConverterLibrary.dll с предыдущего проекта и подключить в текущем проекте.
-2. Создать экземпляр типа Assembly и вызвать метод Load(ConverterLibrary) передав параметром имя библиотеки;
-3. Создать экземпляр типа Type и на экезмпляре класса Assembly вызвать метод GetType(ConverterLibrary.TemperatureConverter) передав параметром класс TemperatureConverter из библиотеки;
-4. Динамически создать экземпляр класса TemperatureConverter на классе-объекте Activator вызвать метод CreateInstance() и параметром передать ссылку на тип Type, который содержит информацию о классе TemperatureConverter;
-5. Вызвать метод ConvertToFahrenheit(25) и передать значение температуры в Цельсиях;
-6. Вывести возвращаемое значение метода на консоль.
