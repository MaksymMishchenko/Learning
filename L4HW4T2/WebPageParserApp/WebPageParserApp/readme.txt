Задание 2
Напишите программу, которая бы позволила вам по указанному адресу web-страницы
выбирать все ссылки на другие страницы, номера телефонов, почтовые адреса и сохраняла
полученный результат в файл.

Решение:
-1. Создать переменную string под строку, в которой будет храниться ссылка;
-2. Создать экземляр класса Regex, в конструктор передать регулярное выражение, которое будет выбирать ссылки, номера телефонов, почтовые адреса;
-3. Создать цикл for, который будет бегать по html-странице и выбирать все необходимы данные;
-4. Создать экземпляр класса StreamWriter и вызвав метод Write() записать найденные данные в файл, закрыть стрим;
-5. С помощью экземпляра StreamReader считать данные с файла и вывести на консоль, закрыть стрим.
