Задание 2
Создайте проект по шаблону "Console Application". Создайте метод «private static void
WriteChar(object symbol)». В теле метода создайте цикл for, размерностью 160 итераций, который
в своем теле с задержкой в пол секунды выводит на экран консоли значение параметра symbol,
приведенного к типу char. Вызовите метод WriteChar из метода Main в контексте задачи,
передавая в качестве параметра значение "!". Все время, пока метод WriteChar выполняется, из
метода Main выводите на экран консоли "$". Когда задача закончит свое выполнение выведите
на экран консоли строку "Метод Main закончил свою работу".

Решение:
+ 1. В теле класса Program создать метод WriteChar(object symbol):
  + Привести переменную типа object к типу char;
  + Создать цикл в 160 итераций, в теле цикла выводить значение переменной symbol;
  + Вызвать статический метод Sleep(500) класса Thread;
+ 2. В методе Main создать асинхронную задачу создав экземпляр класса Task, передать в конструктор метод, вторым параметром передать восклицательный знак;
+ 3. На экземпляре вызвать метод Start();
+ 4. Создать цикл while, в условии которого вызвать на экземпляре свойство IsCompleted, в теле цикла выводить на консоль $;
+ 5. Вывести на консоль, что метод Main() отработал.
