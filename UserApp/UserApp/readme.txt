Задача:
Добавить пользователей в базу данных используя EntityFramework. Вывести на экран информацию об успешном добавлении пользователся в базу данных.

Решение:
+ 1. Уcтановить в проект Microsoft.EntityFrameworkCore.SqlServer и MicrosoftEntityFrameworkCore.Tools.
+ 2. Создать модель User с тремя свойствами (id, имя, возраст).
+ 3. Создать подключение с базой данных наследовавшись от класса DB Context. В классе переопределить метод OnConfigure() и добавить модель User.
+ 4. Используя конструкцию using создать экзекмляр класса подключения к базе данных и добавить пользователей в базу.
+ 5. Вывести на экран пользователей из базы данных.
- 6. Добавить методы UpdateUsers, DeleteUsers (CRUD).
