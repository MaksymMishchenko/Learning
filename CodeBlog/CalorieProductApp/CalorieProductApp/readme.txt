Задача:
Написать программу, которая будет запрашивать у пользователя информацию о продукте,
калорийности и количестве употребленной продукции и записывать данные в файл.
Также в файл должна добавляться информация о дате и времени добавления.
У пользователся должна быть возможность добавить информацию о новом продукте в файл,
а также считать данные с фала и выйти из программы.

Решение:

+1. Создать класс UserProduct, в котором определить три метода:
 + добавить продуктв файл;
 + считать продукт из файла;
 + выйти из программы - завершить работы программы;

+2. В Program в методе Main вывести информацию пользователю о возможном взаимодействии с программой.
+3. Создать экземпляр класса UserProduct.

+4. Создать метод, который  в качестве параметра будет принимать объект типа UserProduct и с помощью конструкции switch case -
 взаимодействоват с ней вызывая той или иной метод в классе UserProduct, в зависимости от выбора пользователя.
5. Обработать исключения, если пользователь введет неправильные данные.
