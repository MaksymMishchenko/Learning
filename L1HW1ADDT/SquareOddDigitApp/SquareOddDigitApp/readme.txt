Создайте метод, который в качестве аргумента принимает массив целых чисел и возвращает
коллекцию квадратов всех нечетных чисел массива. Для формирования коллекции
используйте оператор yield

Решение:
-1. Создать метод GetCollection() который в качестве параметра принимает массив целых чисел и возвращает коллекцию типа IEnumerable;
-2. С помощью цикла for проходимся по каждому элементу коллекции и в условной конструкции if проверяем число на четность.
-3. Если число не четное то с помощью оператора yield return возвращаем коллекцию квадратов всех нечетных чисел.
-4. В методе Main() создаем массив целых чисел и вызвав метод GetCollection() передаем массив.
-5. С помощью цикла foreach перебираем коллекцию и выводи на консоль.
