Задача:

Написать Фабричный метод, который будет возвращать данные типа объекта. С помощщью цикла создать несколько обьектов в помощью метода.

Решение:

+1. Создать класс Rectangle;
+2. В классе создать два поля _x, _y и метода: Factory() и Show();
+3.Метод Factory() создает экземпляр типа Rectangle принимает два параметра, которыми инициализируются поля и возвращает объект Rectangle;
+4. В классе Program создаем экземпляр класса Rectangle, создаем в цикле переменную типа Rectangle и присваиваем ей возвращаемое знаяение метода Factory;
+5. В качестве параметра методов передаем значение счетчиков i и j;
+6. С помощью метода Show() выводим значение обьектов на консоль.
