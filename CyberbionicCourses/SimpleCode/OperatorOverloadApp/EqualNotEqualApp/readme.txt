Пример перегрузки бинарного оператора на равество (==, !=)
Для == и != переопределяются методы Object.Equals() и Object.GetHashCode().

В исходной реализации, по умолчанию Equals() поддерживает сравнение только ссылочных типов.
Поэтому нужно переопределить метод Equals() для сравнения структурных типов.
Переопределение GetHashCode() обязательна при переопределении Equals().
