В программе представлены всевозможные перегрузки операторов, а также правила использования перегрузок операторов.

- Существуют унарные и бинарные перегрузки операторов. Унарные когда в качестве аргумента передается один параметр, бинарные когда два параметра.
- Перегрузка оператора осуществляется вместе с ключевым словом static
- Использовать перегрузку операторов можно ни только снаружи ну и внутри класса посредством методов.
- Есть операторы, которые перегружаются только парно (== !=, < >, <= >=)
- Для == и != переопределяются методы Object.Equals() и Object.GetHashCode().
