Задача:
Написать фабричный метод, который вернет объект класса.

Решение:

+1. Создать класс Rectangle, два поля типа int, проинициализировать поля в пользовательском конструкторе.
+2. Создать метод, который вернет площадь прямоугольника
+3. Создать метод Show(), который покажеш длину и высоту сторон прямоугольника
+4. Создать фабричный метод, который в качестве параметра принимает значение, на которое нужно увеличить длину и высоту прямоугольника и возвращает новый объект "Прямоугольник".
