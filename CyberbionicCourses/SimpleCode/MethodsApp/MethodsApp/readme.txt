Задача:

Написать метод для поиска индекса элемента массива [тип элементов в массиве - int].
Метод должен вернуть индекс первого найденного элемента, если он будет найден.

Решение:
- 1. Написать тесты для методов;
+ 2. Создать класс и методы:
     + метод который возвращает массив;
     + метод, который рандомно заполняет числами массив;
     + метод, который ищет число в массиве и возвращает под каким оно индексом, если есть;
     + метод, который обрабатывает результат полученного индекса и выводит его на консоль.
