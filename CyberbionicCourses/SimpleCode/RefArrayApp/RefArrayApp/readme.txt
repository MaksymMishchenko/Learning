Задача:
Написать generic-метод Resize изменяющий количество элементов массива.
(метод должен иметь возможность увеличить или уменьшить количество элементов массива) 

Решение:
+ 1. Создать метод, который в параметрах будет принимать два значения - сам массив с оператором ref, и значение типа int - размерность массива.
+ 2. В методе создаем новый массив на указанное пользователем количество элементов.
+ 3. С помощью цикла for перемещаем значения со старого массива в новый
+ 4. В качестве старого массива возвращает ссылку на новый.
