Задача:
Создать сервис, который запрашивает стоимость акции банка Америки и выводит ее на консоль. Использовать делегат при создании сервиса.

Решение:
+1. Создать класс StockExchangeMonitor, в котором обьявить делегат и свойство типа делегата, в которое передадим ссылку на метод;
+2. В StockExchangeMonitor создать метод Start, который с помощью рандомайзера будет генерить стоимость акции;
+3. Создать экземпляр делегата, создать метод, который будет выводить сообщение на консоль и передать в делегат;
+4. Вызвать метод Start.
