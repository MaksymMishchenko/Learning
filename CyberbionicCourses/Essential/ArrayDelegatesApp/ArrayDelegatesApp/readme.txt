Задание 
Используя Visual Studio, создайте проект по шаблону Console Application.  
Создайте анонимный метод, который принимает в качестве аргумента массив делегатов и
возвращает среднее арифметическое возвращаемых значений методов сообщенных с делегатами в массиве.
Методы, сообщенные с делегатами из массива, возвращают случайное значение типа int. 

Решение:
+1. Создать класс-делегат Delegate1 который в качестве параметра будет принимать массив делегатов и
возвращать в качестве возвращаемого значения среднее арифметическое возвращаемых значений методов сообщенных с делегатами в массиве;
+2. Создать анонимный метод, который будет сообщен с Delegate1 который будет принимать в качестве параметра массив методов сообщенных с делегатом и возвращать среднее арифметическое методов сообщенных с делегатом;
+3. Создать класс-делегат Delegate2 который будет сообщен с методом, который в качеств епараметра будет принимать 1 аргумент-массив и возвращать значение типа int;
+4. Создать метод GetRandom без аргументов и с возвращаемім значением int и сообщить его с делегатом Delegate2;
+5. Вывести значение на экран.
