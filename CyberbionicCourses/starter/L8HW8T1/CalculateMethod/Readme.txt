The task
Using Visual Studio, create a project using the Console Application template.
Create a method named Calculate that takes three integer values ​​as parameters and returns the value of each argument divided by 5.

Solution:
1. Let's create a method called Calculate, which returns Void by specifying the "ref" parameter and the "int" type in the method signature. In the body of the method, divide each variable by 5.
2. In the Main method, we create three integer variables and assign values ​​to them. We call the Calculate method and pass our variables.
3. We display the result of dividing our variables by 5.
