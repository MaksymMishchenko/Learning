Задание 3
Напишите приложение для поиска заданного файла на диске. Добавьте код, использующий
класс FileStream и позволяющий просматривать файл в текстовом окне. В заключение
добавьте возможность сжатия найденного файла.

Решение:
+1. Разместить на форме окно отображения логических дисков, окно просмотра файла, кнопки - Поиск, Архивировать, Просмотреть.
+2. Создать метод SearchFile, в котором на классе-объкте получить все логические диски устройства DriveInfo.GetDrives(). На экземпляре класса FileInfo получить все файлы методом GetFiles();
+3. Обратиться к выбранному логическому диску и произвести поиск файла.
+4. По нажатию на кнопку Просмотреть - создать метод просмотра найденного файла. В методе создать экземпляр StreamReader и передать к него путь к файлу. Прочитать файл до конца, закрыть стрим;
+5. По нажатию на кнопку Архивировать создать метод, в котором указать путь к файлу, указать путь создания архива. Передать в экземпляр класса GZipStream путь создания архива и вторім параметром CompressionMode.Compress. Закрыть компрессор.
