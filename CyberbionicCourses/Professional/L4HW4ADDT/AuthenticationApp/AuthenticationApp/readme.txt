Задача:
Напишите консольное приложение, позволяющие пользователю зарегистрироваться под
«Логином», состоящем только из символов латинского алфавита, и пароля, состоящего из
цифр и символов.

Решение:
+1. Создать класс User, который содержит свойства Id, Name, Password;
+2. Создать List<User> закрыть типом User;
+3. Запросить у пользователя ввести имя и пароль с консоли;
+4. Проверить, ввел ли пользователь логин состоящий из латинского алфавита и пароль c помощью регулярных выражений и класса Regex;
+5. Если проверка пройдена успешно - зарегистрировать пользователя, добавив данные в List<User> вызвав на экземпляре метод Add;
+6. Если полльзователь ввел недопустимые символы - вывести сообщение об ошибке.
