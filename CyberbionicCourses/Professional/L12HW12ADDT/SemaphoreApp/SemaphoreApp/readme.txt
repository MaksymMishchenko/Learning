Дополнительное задание
Создайте Semaphore, осуществляющий контроль доступа к ресурсу из нескольких потоков.
Организуйте упорядоченный вывод информации о получении доступа в специальный *.log
файл.

Решение:
+1. Создать переменную Semaphore pool для доступа к ресурсу из нескольких потоков;
+2. Создать метод LogFile(object name), который блокирует доступ к файлу других потоков poll.WaitOne(),
    внутри создает log-файл и в конце будит основной поток вызвав метод Release();
+3. В методе Main() создать экземпляр класса Semaphore(1, 3, "MySemaphore");
+4. Создать цикл на 8 итераций и запустить в цикле потоки, передав метод LogFile;
+5. Методу Start класса Thread передать счетчик итерации - Start(i).
