Задание:
Используя класс SortedList, создайте небольшую коллекцию и выведите на экран значения пар
«ключ- значение» сначала в алфавитном порядке, а затем в обратном.

Решение:
+1. Создать коллекцию SortedList;
+2. Методом Add добавить несколько элементов ключ-значение в коллекцию;
+3. С помощью foreach вывести элементы коллекции ключ-значение на консоль;
+4. Создать класс InsensitiveComparer, который реализует интерфейс IComparer;
+5. В классе создать экземпляр класса CaseInsensitiveComparerDescending и на экземпляре вызвать метод Compare, в который передать параметры y и x
+6. Передать экземпляр класса InsensitiveComparer в конструктор коллекции SortedList;
