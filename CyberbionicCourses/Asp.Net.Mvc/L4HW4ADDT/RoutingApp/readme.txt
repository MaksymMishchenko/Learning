Задание 
Сделайте метод действия, который будет выдавать в ответ пользователю представление с сообщением «данные не предоставлены» если запрос сделанный к контроллеру не содержит параметра id переданного через сегменты. Если параметр есть, метод действия должен возвращать строку (plain/text) со значением, которое находится в сегменте id. 

Решение:
1. В методе действия Index() принимать параметр id типа string;
2. Сделать проверку, если параметр null, тогда вернуть ошибку 404 NotFound, иначе вернуть тип Content();
3. Создать две ссылки - с параметром и без, проверить работу контроллера.
