Задание 2
Создайте коллекцию, в которой бы хранились наименования 12 месяцев, порядковый номер и
количество дней в соответствующем месяце. Реализуйте возможность выбора месяцев, как по
порядковому номеру, так и количеству дней в месяце, при этом результатом может быть не
только один месяц.

Решение:
+1. Создать класс Calendar, который реализует интерфейс IEnumerable, IEnumerator;
+2. В классе создать два массива: months - для количества месяцев в году и days - для количества дней в месяце; 
+3. Реализовать члены интерфейсов Current, MoveNext, Reset, GetEnumerator;
+4. Запросить от пользователя ввести месяц;
+5. Циклом foreach вывести на консоль месяц и количество дней в месяце;
-6. В классе Calendar создать метод, который будет выводит информацию по месяцам;
-7. В классе Calendar создать метод, который будет выводит информацию по дням.
