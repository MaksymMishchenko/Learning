Задание 2
Создайте коллекцию, в которую можно добавлять покупателей и категорию приобретенной ими
продукции. Из коллекции можно получать категории товаров, которые купил покупатель или по
категории определить покупателей.

Решение:
-1. Создать класс Customer, который содержит свойство LastName и его значение инициализируется в конструкторе значением аргумента;
-2. Создать enum перечисление Category, которое содержит список категорий;
-3. В методе Main() создать коллекцию Dictionary<Customer, Category>, заполнить коллекцию элементами;
-4. Создать два метода - GetCategoryByCustomer и GetCustomerByCategory;
-5. Метод GetCategoryByCustomer первым параметром принимает коллекцию, вторым покупателя;
-6. Метод GetCustomerByCategory первым параметром принимает коллекцию, вторым категорию;
-7. Вызвать методы GetCategoryByCustomer, GetCustomerByCategory
