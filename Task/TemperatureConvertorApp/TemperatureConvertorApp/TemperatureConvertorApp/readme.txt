Задача:
Температуру, измеренную в градусах по Цельсию, можно перевести в градусы по Фаренгейту путем умножения на 9/5 и сложения с числом 32.
Напишите программу, запрашивающую температуру в градусах по Цельсию и отображающую ее эквивалент по Фаренгейту.

Решение:
- 1. Пишем тест и проверяем с его помощью соответсвует ли наша программа требованиям.
- 2. В классе Program в методе Main() выводим на консоль сообщение пользователю о необходимости ввести температуру в градусах по Цельсию.
- 3. Создаем класс TemperatureConvertor, в нем метод CelsiusToFahrenheit(), который будет возвращать результат по Фаренгейту.
- 4.  В классе Program в методе Main() создаем экземпляр нашего класса TemperatureConvertor. На экземпляре вызываем наш метод CelsiusToFahrenheit(). Присваиваем возвращаемое значение переменной result/
- 5. Выводим результат на экран.
