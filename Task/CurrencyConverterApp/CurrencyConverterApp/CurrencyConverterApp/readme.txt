Задача:
На биржевых торгах за 1 фунт стерлингов давали $1.487, за франк — $0.172,
за немецкую марку — $0.584, а за японскую йену — $0.00955. Напишите
программу, которая запрашивает денежную сумму в долларах, а затем выводит эквивалентные суммы в других валютах.

Решение
+ 1. Пишем тест, который будет проверять, соответствует ли возвращаемое значение наших методов требованию.
+ 2. Просим пользователя ввести денежную сумму в долларах.
+ 3. В теле класса Program создаем переменную типа float, и присваиваем ей значения введенное пользователем в консоли. 
+ 4. Создаем клас ExchangeRate. В нем создаем константы pound, franc, mark, yen типа float.
+ 5. Создаем класс CurrencyConvertor, в котором создаем поле типа _dollar и 4 метода, каждый из которых принимает значение от пользоватеkя и возвращает результат конвертации по каждой валюте. В методах подставляем соответствующую константу из класса ExchangeRate.
+ 6. В классе Program в методе Main() создаем экземпляр класса CurrencyConvertor.
+ 7. На экземпляре вызываем методы, которые отвечают за конвертацию необходимой валюты. Выводи результат по каждому курсу валют на экран.
