Задача:
Создать раздаляемый ресурс - метод, в котором будут суммироваться элементы массива. 
Создать два потока, которые обращаются к разделяемому ресурсу и возвращают результат суммирования элементов массива.
Устроить доступ к разделяемому ресурсу, чтобы два потока поочередно обращались к ресурсу.

Решение:
- 1. Создать класс SumArray, создать объект, который синхронизируем доступ к ресурсу;
- 2. В классе SumArray создать метод GetSum();
- 3. Создать класс MyThred, в нем поле типа массива - int[] arr и типа Thread - Threat _thread и int answer;
- 4. Создать экземпляр класса SumArray;
- 5. В конструкторе проинициализировать int[] arr и Threat _thread(создать новый поток). Даем потоку имя, и запускаем поток
- 6. Создать метод Run без параметров, который передать в поток;
- 7. Метод Run содержит сообщение о том, что поток начал свою работу. Полю answer присвоить возвращаемое значение метода GetSum(),вызванного на экземплре класса SumArray;
- 8. Вывести на консоль результат поля answer. Сообщить о завершении метода.
- 9. В классе Program создать массив. Создать экземпляр класса SumArray и в метод GetSum() передать массив.
- 10. Создать два потока, в конструкторе дать имена потокам.
- 11. С помощью метода Join обратиться к двум потоком чтобы узнать состояние потоков.
