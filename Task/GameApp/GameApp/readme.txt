Задача:

Создать игрового персонажа, который метает оружием. При разработке использовать паттерн проектирования Стратегия.

Решение:

- 1. Создаем интерфейс IWeapon, в нем метод Shoot(); 
- 2. Создаем виды оружия для нашего персона и реализуем интерфейс IWeapon;
- 3. Создаем класс Hero, поле типа интерфейсного типа и имени, задаем имя в конструкторе;
- 4. Создаем метод SetWeapon в классе Hero, в параметре принимаем тип оружия;  
- 5. Создаем метод Shoot в классе Hero и стреляем оружием.
- 6. В классе Program создаем героя, даем ему имя, оружие и стреляем.
