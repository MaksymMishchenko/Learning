Задача:
Добавить людей в коллекцию и вывести с коллекции на экран в представлении, как на примере ниже

Фамилия Имя Адрес Город
-------------------------------------------------------------------
Петров Василий Кленовая 16 Санкт-Петербург
Иванов Сергей Осиновая 3 Находка
Сидоров Иван Березовая 21 Калининград

Решение:

+ 1. Создаем класс Person, в котором будет находится наша модель.
+ 2. В Program в методе Main() создаем  коллекцию List и закрываем типом Person
+ 3. Добавляем новых людей в коллекцию.
+ 4. С помощью цикла foreach извлекаем данные из коллекции и передаем в класс ShowPerson в метод Show().
+ 5. C помощью табуляции выводим данные на экран согласно примеру в задаче.
