Задача:
Написать программу, которая будет прибавлять значения фунтов и дюймов и выводить результат операции расчета на экран.

Решение:
 + 1. создать тесты для метода, который будет расчитывать результат добавления фунтов и дюймов;
 + 2. создать структуру с двумя полями для футов и дюймов, проинициализировать значение полей в конструкторе;
 - 3. с помосчью блока инициализации присвоить некоторые значения нашим полям
 + 4. создать метод, который будет принимать две переменные фунты и дюймы введенные пользователем и возвращать результат операции сложения значения полей и пользовательского ввода значений;
 - 5. вывод результата сложения на экран.
