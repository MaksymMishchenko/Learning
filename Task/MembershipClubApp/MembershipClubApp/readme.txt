Задача:

Создать три бизнес-сущности, абонементы в спортзал, которые будут содержать информацию о имени абонемента, описании и цене. При разработке использовать паттерн Фабричный метод.

Решение:

+ 1. Создать общий интерфейс для всех бизнес-сущностей, в котором определить имя, описание и цену.
+ 2. Реализовать интерфейс классами-абонементами, в которые добавить два поля для описания и цены. А имя абонемента проинициализируем в кострукторе каждого класса.
+ 3. Создать абстрактную фабрику в которой создать абстрактный фабричный метод.
+ 4. Создать три конректные фабрики, которые наследуются от абстрактной фабрики и переопределить фабричный метод. Кроме толо добавить поля цены и описания.
+ 5. В классе Program написать приветсвие и с помощью switch дать возможность оператору выбора абонемента.
+ 6. Создать метод, который будет возвращать абонемент в зависимости от выбора оператора.
+ 7. Возвращаемой значение конкретной фабрики присвоить переменной типа абстрактной фабрики.
+ 8. Переменной интерфейса абстрактной фабрики присвоить возвращаемое значение абстрактного метода.
+ 9. Вывести на консоль информацию овыбранном абонементе.
