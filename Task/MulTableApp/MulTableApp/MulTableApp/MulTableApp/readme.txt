Задача:
Вывести табличку умножения в столбик от 2 до 10. Обвести выведенное значение в табличку.

Решение:
Этап 1: готово
1. Создать метод, который отрисует таблицу в консоли с помощью метода Console.WriteLine();
2. Создать строкe и передать в наш метод табличку умножения на 2x2=4.
3. С помощью цикла for вывести на экран табличку, в которой разместить данные из массива строк

Этап 2: готово
1. Создать массив строк таблицы умножения на 2 и передать аргументом в наш метод, который отрисовывает табличку. Вывести табличку умножения на 2 на экран с помощью цикла for.
2. Создать два массива строк таблицы умножения на 2 и 3.
3. Вывести табличку умножения на 2 и 3 на экран с помощью цикла for.
4. Разделить колонки между собой, чтобы каждая колонка была в отдельной ячейке общей таблицы.

Этап 3: =>в работе

1. Создать массив массивов, в который поместить 3 наших столбика на 2x, 3x, 4x.
2. C помощью цикла for к первым двум столбикам на 2x и 3x вывести третий столбик 4x на экран.
3. Разделить столбик 3x и 4x между собой.
4. Вывести табличку умножения на 2x, 3x, 4x. на экран с помощью цикла for.
