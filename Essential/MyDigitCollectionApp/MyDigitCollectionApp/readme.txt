Задание 
Используя Visual Studio, создайте проект по шаблону Console Application.  
Создайте метод, который в качестве аргумента принимает массив целых чисел и возвращает коллекцию всех четных чисел массива. Для формирования коллекции используйте оператор yield.

Решение:

- 1. Создать класс Digit;
- 2. В классе создать метод GetEnumerator(int[] arr), который принимает массив целых чисел и вернет коллекцию;
- 3. В конструкции if указать выборку по четным числам из коллекции и с помощью операторов yield return вернуть элементы;
- 4. Создать массив на 10 элементов, заполнить числами;
- 5. Создать экземпляр класса Digit. НА экземпляре вызвать метод, в который передать массив чисел; 
- 6. Перебрать коллекцию элементов с помощью цикла foreach.
