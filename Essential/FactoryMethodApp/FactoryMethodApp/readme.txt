
Задание 
Используя Visual Studio, создайте проект по шаблону Console Application.  
Создайте класс MyClass<T>, содержащий статический фабричный метод -  T FacrotyMethod(), 
который будет порождать экземпляры типа, указанного в качестве параметра типа (указателя места заполнения типом – Т). 

Решение:
-1. Создать класс MyClass<T> параметризированный местом заполнения типом Т;
-2. Создать параметризированный метод в классе MyClass<T>, который будет порождать экземпляры типа;
-3. Создать сообственный тип Animal;
-4. Создать экземпляр класса Animal;
-5. Создать экземляр типа MyClass<T> и закрыть его типом Animal.
