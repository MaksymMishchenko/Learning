Задание 
Используя Visual Studio, создайте проект по шаблону Console Application.  
Создайте четыре лямбда оператора для выполнения арифметических действий: (Add – сложение, Sub – вычитание, Mul – умножение, Div – деление).
Каждый лямбда оператор должен принимать два аргумента и возвращать результат вычисления.
Лямбда оператор деления должен делать проверку деления на ноль.  
Написать программу, которая будет выполнять арифметические действия указанные пользователем. 

Решение:
+1. Создать делегат который принимает два целочисленных пареметра и возвращает int;
+2. Создать экземпляр класса-делегата и сообщить с ним 4 лямбда-оператора;
-3. С помощью метода Console.ReadLine принять от пользователся два целочисленных числа и операцию над числами.
-4. В конструкции Switch/Case вызываем соотвествующий оператор;
-5. Выводим результат операции на экран.
