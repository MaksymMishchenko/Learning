Задание:

Используя Visual Studio, создайте проект по шаблону Console Application.  
Создайте расширяющий метод: public static T[] GetArray<T>(this MyList<T> list) 
Примените расширяющий метод к экземпляру типа MyList<T>, разработанному в домашнем задании 2 для данного урока. Выведите на экран значения элементов массива, который вернул расширяющий метод GetArray(). 

Решение:
- 1. Создаем статический класс и помещаем в него статический метод public static T[] GetArray<T>(this MyList<T> list)
- 2. С помощью foreach в методе-расширении выводим коллекцию на экран.
- 3. На экземпляре MyList<T> вызываем расширяющий метод,который выводит элементы коллекции на консоль.
