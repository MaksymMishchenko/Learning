Задание 
Используя Visual Studio, создайте проект по шаблону Console Application.  
Создайте класс MyDictionary<TKey,TValue>.
Реализуйте в простейшем приближении возможность использования его экземпляра аналогично экземпляру класса Dictionary (Урок 6 пример 5).
Минимально требуемый интерфейс взаимодействия с экземпляром, должен включать метод добавления пар элементов,
индексатор для получения значения элемента по указанному индексу и свойство только для чтения для получения общего количества пар элементов. 

Решение:
-1. Создать класс-дженерик MyDictionary<T>;
-2. В классе создать поле-массив c указателем места заполнения типом Т;
-3. Создать индексатор для получения значения элемента,
-4. свойство только для чтения
-5. Методы:
   - Add;
   - Remove. 
-6. Создать несколько экземпляров класса MyDictionary<T> и закрыть его разными типами.
-7. Добавить, удалить значение в/из коллекции, получит общее количество пар элементовб получить значение из коллекции по указанному индексу.
