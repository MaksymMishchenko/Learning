Дополнительное задание
Создайте массив чисел размерностью в 1 000 000 или более. Используя генератор случайных
чисел, проинициализируйте этот массив значениями. Создайте PLINQ запрос, который
позволит получить все нечетные числа из исходного массива.

Решение:
-1. Создать массив типа int на 1 000 000 элементов;
-2. Создать экземпляр класса Random;
-3. Cоздать обьект типа ParallelLoopState<int> state передав в лямбда-выражение заполнение массива, вызвать метод Next(-100, 100);
-4. Заполнить массив вызвав на классе-обьекте Parallel метод For(0, arr.Lenght, state);
-5. Сформировать запрос возвращаемім значением которого будет ParallelRequest<int> oddNumbers:
 - from item in oddNumbers.AsParallel()
   where item < 0
   select item;
-6. Вывести полученные значения вызвав на классе-объекте Parallel.Foreach()
