Задание:
Создайте пользовательский тип (например, класс) и выполните сериализацию объекта этого
типа, учитывая тот факт, что состояние объекта необходимо будет передать по сети.

Решение:
-1. Создать класс Student с полями: Id, имя, возраст, група и один метод ShowInfo(); проинициализировать поля в конструкторе;
-2. Декорировать класс Student атрибутом [Serializable];
-3. В Program создать экземпляр класса Student, в конструкторе передать значения;
-4. Вызвать метод ShowInfo(), который выводит информацию на консоль;
-5. Создать экземпляр класса MemoryStream;
-6. Создать экземпляр BinaryFormatter, на экземпляре вызвать метод Serialize() для сериализации;
-7. В метод Serialize() первым параметром передать экземпляр MemoryStream, вторым экземпляр класса Student; 
-8. Создать переменную типа bytes и присвоить ей возвращаемое значение метода ToArray вызванном на экземпляре MemoryStream;  
-9. Вызвать метод DeSerialize() и десериализовать состояние полей из обьекта MemoryStream;
