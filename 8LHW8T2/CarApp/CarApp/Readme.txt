Задание 2
Создайте класс, поддерживающий сериализацию. Выполните сериализацию объекта этого
класса в формате XML. Сначала используйте формат по умолчанию, а затем измените его
таким образом, чтобы значения полей сохранились в виде атрибутов элементов XML.

Решение:
+1. Создать класс Car, в нем 4 свойства: CarBrand, Model, Price, MaxSpeed;
+2. Создать экземпляр класса Car, заполнить свойства значениями;
+3. Создать экземпляр класса XmlSerializer(typeof(Car)), в конструктор передать тип Car;
+4. На экземпляре XmlSerializer вызвать метод Serialize() для сохранения состояния полей по умолчанию;
+5. Для сохранения значения полей в XML в виде атрибутов применяем атрибут XmlAttribute к свойству;
