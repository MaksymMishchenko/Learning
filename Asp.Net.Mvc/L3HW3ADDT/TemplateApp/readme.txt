Задание 
Создайте сайт состоящий из 4 представлений: Home, Products, Services, About.
Каждое преставление должно использовать компоновку.
Компоновка должна содержать меню со ссылками на каждое представление и футер с данными об авторе сайта. 

Решение:
1. Создать один контроллер HomeController, в нем 4 метода;
2. Создать 4 представления, добавить секции с ссылками в каждое представление;
3. В файле _Layout подключить секции в соответствующие блоки кода вызвав метод RenderSection();
4. Создать коллекцию продуктов на основе модели Product, передать коллекцию как частичное представление и вызвать из представления Product метод, который вернет продукты;
5. Настроить роутинг, указав, что по умолчанию вызывает метод действия Home из контроллера Home/
