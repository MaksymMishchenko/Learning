Задание 
Создайте веб приложение с одним представлением. В представлении добавьте форму с двумя полями ввода и четырьмя кнопками – «+», «-», «*», «/». Реализуйте контроллер, который будет выполнять соответствующие арифметические операции при нажатии по кнопкам.

Решение:
-1. Создать контроллер Home с методом Index, который возвращает представление;
-2. Создать метод Add, Sub, Mul, Div, пометить атрибутом HttpPost. Методы будуть вызываться в зависимости от нажатой кнопки, реализовывать соответсвующую логику и возвращать представление Index;
-3. В представлении Index создать форму ив ней два input используя Html tag hepler. Добавить 4 кнопки.
-4. Вывести результат операции в теге p.
