Задача:

Создать десктоп-приложение, которое будет хранить список повседневных дел с возможностью отметить задание как выполненное. Должна отображаться дата выполнения задание без возможности редактирования.

Решение:
+1. Создать разметку окна, поделив его на три колонки. Дать название каждой колонке, прописать свойство IsReadonly для каждой колонки.
+2. Создать класс, который будет содержать модель нашего листа.
+3. Создаем лист и закрываем моделью.
