Задание 2
Создайте файл, запишите в него произвольные данные и закройте файл. Затем снова откройте
этот файл, прочитайте из него данные и выведете их на консоль.

Решение:
-1. Создать файл с помощью экземпляра классаFileInfo, в конструктор передать путь к файлу.
-2. На экземпляре вызвать метод Create();
-3. Создать экземпляр класса StreamWriter и на экземпляре вызвать метод Write(), в качестве параметра передать текст;
-4. Закрыть стрим StreawWriter;
-5. Создать экземпляр класса StreamReader в качестве параметра конструктора передать путь к файлу для чтения;
-6. Создать переменную типа string и присовить ей возвращаемое значение метода ReadToEnd, вызванном на экземпляре StreaReader;
-7. Закрыть стрим StreamReader;
-8. Вывести результат на консоль.
