Задание
Создайте проект по шаблону "WPF". Переместите из элементов управления (ToolBox) на форму
текстовое поле (TextBox) и кнопку (Button). Дайте имена для ваших элементов управления, чтобы
к ним можно было обращаться из кода. Например, текстовое поле – txtResult, а кнопка – btnStart.
Создайте и зарегистрируйте обработчик события по нажатию на кнопку btnStart. Он должен в
цикле выводить в текстовое поле звездочки с задержкой в 300 миллисекунд. В текстовое поле
должно быть выведено 100 звездочек. Чтобы форма не зависла на время вывода звездочек и
могла отвечать на действия пользователя реализуйте выполнение с помощью задач.

Решение:
+ 1. Создать метод WriteStar() который в цикле выводит звездочки в текстовую область.
+ 2. Создать обработчик события по нажатию на кнопку Start.
+ 3. С помощью статического метода Run() класса Task использовать лямбда-вырежение и запустить метод WriteStar() в контексте вторичного потока.
