Задача:

Создать игрока, который будет использовать разные виды оружия, а также степенью нанесенного урона.
Всю информацию отображать на консоли, в т.ч о том, какое оружие было использовано.

Решение:
+1. Создать абстрактный класс Weapon, в котором создать абстрактный метод Fire(), абстрактное свойство, которое содержит уровень урона и метод, который будет выводить информацию об оружии.
+2. Создать три класса с оружием - пистолет, автомат, лук и отнаследоваться от абстрактного класса Weapon.
+3. Создать класс игрока, который будет иметь метод Fire() и принимать в качетсве параметра оружие типа Weapon.
+4. В классе Program в методе Main создать экземпляр класса Player.
+5. Создать массив типа Weapon и добавить в него все виды нашего оружия.
+6. С помощью цикла foreach вывести на экран вид оружия, которым может пользоваться игрок, также уровень урона нанесенное определенным оружием.
+7. Реализовать написанную программу с помощью интерефейсов.
