Задача:
Дан заполненный массив заполненный случайными числами на 15 элементов.
Пройтись по массиву с помощью цикла for и получить сумму всех четных чисел в массиве. Вывести результат на экран.

Решение: 
 1. Написать тесты для методов.
- 2. Создать класс с тремя методами:
     + первый - будет создавать массив;
     + второй - заполнять его случайными числами;
     + третий - возвращает сумму всех четных чисел в массиве;
     + четвертый - выводит значения массива на консоль.
