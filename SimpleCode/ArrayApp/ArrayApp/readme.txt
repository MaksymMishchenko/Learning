Задача:
Заполнить массив с клавиатуры с помощью цикла for и вывести значение на экран

Решение: 
- 1. Написать тесты для метода, который создает и заполняет массив.
- 2. Запросить у пользователя данные с консоли:
     - на сколько элементов будет массив;
     - начальное число заполнения массива; 
- 2. Создать класс с тремя методами:
     - первый - будет создавать массив;
     - второй - заполнять его пользовательскими значениями;
     - третий - выводить значения массива на консоль.
