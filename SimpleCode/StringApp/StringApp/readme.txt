Задача:

Написать программу, которая будет принимать от пользователя целое число и выводить значение числа буквами на консоль.

Решение:

+1. Создать два массива - string[] и int[];
+2. В массив строк добавить элементы буквами;
+3. С помощью цикла do while расчленить число введеное пользователем с помощью операторов "%", "/" и добавить в массив int[];
+4. Вывести элементы массива string[] указав в скобках параметром массив int[];
