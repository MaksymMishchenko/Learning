Задача:

Написать программу, которая в зависимости от уровня ошибки будет выводить на консоль сообщение определенным цветом. Также создать список работников, при котором возникла та или иная ошибка.
Для списка работников и списка ошибок воспользоваться enum. Реализовать метод работников в интерфейсе по умолчанию.


Решение:

-1. Создаем enum Employees и перечисляем сотрудников;
-2.Создаем enum LogLevel и перечисляем наши ошибки;
-3.-1.Создаем интерфейс IConsoleLogger с одним методом Log, который будет принимать структура LogLevel, а также сообщение;
-4.Создаем класс ConsoleLogger, реализуем интерфейс IConsoleLogger; C помощью конструктции switch в зависимости от уровня ошибки, изменяем цвет текста консоли;
-5.Выводим время возникновения ошибки на консоль и сам текст;
-6.Сбрасываем цвет текста;
-7.Создаем экземпляр класса типа ссылки IConsoleLogger.
-8.На экземпляре вызываем метод Log(), передаем уровень ошибки и текст.
-9.В интерфейсе IConsoleLogger создаем метод ShowEmployee(), который принимает тип Employees и тут же реализовываем его выводя на консоль имя сотрудника
