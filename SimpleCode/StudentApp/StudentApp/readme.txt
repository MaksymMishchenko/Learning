Задача:

Создать обьект студент, который имеет Id, Имя, Фамилия, Отчество, Возраст, Группу.

Решение:

- 1. Создать класс Student со свойстами Id, Имя, Фамилия, Отчество, Возраст, Группу.
- 2. Создать метод GetStudent з возвращаемым значением типа Student, который в качестве параметра принимает студента типа Student.
- 3. Создать в методе экземпляр класса Student и присвоить значения соответсвенным полям.
- 4. Создать метод PrintStudent, который в качестве параметра принимает студента типа Student.
- 5. Вывести значения полей на экран поместив в метод WriteLine().
- 6. В Program создать переменную типа Student и присвоить ей возвращаемое значение метода. 
- 7. Вызвать метод PrintStudent и в качестве параметра передать переменную, которой ранее присвоено возвращаемое значение метода GetStudent.
