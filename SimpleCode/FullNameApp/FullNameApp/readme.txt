Задача:

Создать проект WinForms, который будет выводить список созданных нами персон, а именно Фамилия, Имя, Отчество.
Если в колонке Отчество нет данных, использовать  оператор null-обьединения и вывести соответсвующее сообщение на экран.


Решение:

+ 1. При создании проекта выбрать WinForms.
+ 2. Нанести на форму ListBox и Button.
+ 3. Определить модель в классе Person (свойства) и метод, который будет выводить список персон
+ 4. В форме создать список и метод, который будет пробегаться по коллекции персон.
+ 5. В классе Person воспользоваться оператором обьединения ?? и после него вывести соответствующее сообщение.
+ 6. Добавить контакты в класс Person.
