Задача:

Написать программу, которая будет работать по принципу стека. В качестве хранения данных в стеке служит закрытый массив.

Решение:
+1. Создать класс Stack, поместить два приватных поля: типа массива и индекса массива;
+2. Создать метод, который будет добавлять элементы в стек;
+3. Создать метод, который будет извлекать элементы из стека;
+4. Создать две булевые переменные которые будут возвращать значение (true/false) полный или пустой наш массив;
+5. Создать метод, который будет проверять размер нашего массива;
+6. Создать метод, который будет проверять, размерность нашего массива;
+7. Создать три экземпляра класса Stack;
+8. Создать метод, который заполнит массив в первом экземпляре;
+9. Создать метод, который выведет элементы массива на экран из первого экземпляра;
+10. Создать метод, который из первого экземпляра добавит данные во второй экземпляр;
+11. Создать метод, который заполнит третий экземпляр количеством в 5 элементов;
+12. Создать метод, который выведет размерность всего масива третего экземпляра;
+13. Создать метод, который выведет количество элементов в третем массиве.
