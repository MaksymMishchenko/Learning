Задача:
Написать методы для добавления элемента в начало массива, конец массива и по указанному индексу.

Решение:
+ 1. Создать три метода:
  + Метод Insert, который будет принимать три параметра - массив, индекс и значение. В теле метода создаем новый массив и перезаписываем его старыми значениями.
    Присваиваем по указанному индексу значение, которое нужно поместить в массив.
    С помощью цикла перебираем значения старого масива и записываем в новый. В условии указываем, что перебираем лишь до идекса указанного пользователем.
    С помощью второго цикла перебираем оставшиеся значениям в старом массиве со сдвигом + 1.

  + Метод InsertFirst, который будет принимать два параметра - массив и значение, которое нужно поместить в первый индекс;
  + Метод InsertLast, который будет принимать два параметра - массив и значение, которое нужно поместить в последний индекс.
