Задача:
Написать методы для удаления первого элемента массива, последнего элемента массива и элемента по указанному индексу.

Решение:
- 1. Создать три метода:
  - Метод RemoveAt, который будет принимать два параметра - массив, индекс. В теле метода создаем новый массив и перезаписываем его старыми значениями.
    С помощью цикла перебираем значения старого масива и записываем в новый. В условии указываем, что перебираем лишь до идекса указанного пользователем.
    С помощью второго цикла перебираем оставшиеся значениям в старом массиве со сдвигом - 1.
    Обходим элемент массива по индексу, который нужно удалить при заполнении нового массива значениями со старого.

  - Метод RemoveAtFirst, который будет принимать два параметра - массив и значение, которое нужно поместить в первый индекс;
  - Метод RemoveAtLast, который будет принимать два параметра - массив и значение, которое нужно поместить в последний индекс.
