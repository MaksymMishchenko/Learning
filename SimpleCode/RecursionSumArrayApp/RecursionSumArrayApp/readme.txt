Задача: 

Найти сумму элементов массива с помощью рекурсии.

Решение:

+ 1. Создать массив на 5 элементов.
+ 2. Создать рекурсивный метод, который будет принимать в качества параметров наш массив и один необязательный параметр, который будет использовать как счетчик.
+ 3. В методе с помощью конструкции If продумать выход из рекурсивного метода.
+ 4. Увеличивая счетчик, пробегаемся по каждому элементу массива и прибавляем в переменную result;
