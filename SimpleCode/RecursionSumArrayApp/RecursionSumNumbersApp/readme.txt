Задача: 

Найти сумму цифр числа с помощью рекурсии. 561 = 12

Решение:

- 1. Создать переменную типа int и присвоить значение 561 .
- 2. Создать рекурсивный метод, который будет принимать в качества параметра переменную с числом.
- 3. В методе с помощью условия продумать выход из рекурсивного метода.
- 4. с помощью операторов деления и деления с остатком разбить целое число на числа и сложить их вместе.
