Задача:

Написать программу простого калькулятора, который выполняет 4 арифметические действия над оперfндами: +, -, *, /. Предусмотреть попытку деления на 0, вывести соостветсвующее сообщение.
Обработать попытку ввода пользователя вместо цифр других символов и вывести соответсвующее сообщение. Сделать программу с помощью конструкции SWITCH и IF/ELSE/

Решение:
- 1. Написать тесты для методов, которые производят действия над двумя операндами: суммирует, отнимает, умножает, делит.
- 2. Создать класс, поместить 4 метода.
- 3. Принимаем от пользователя два операнда, конвертируем в int.
- 4. С помощью конструкции try/catch проверяем, не ввел ли пользователь сиволы отличные от цифр. Если ввел, выводим сообщение, что пользователь ввел неверные данные.
- 5. С помощью конструкции IF/ELSE проверяем условие - операцию, которую ввел пользователь над операндами.
- 6. Выводим результат произведенный над двумя операндами на экран.
