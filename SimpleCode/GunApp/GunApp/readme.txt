Задача:

Написать программу GunApp, которая будет иметь private-метод, который будет перезарежать оружие, и public-метод Shoot, который будет использовать данный метод.

Решение:
+1. Создать класс Gun, в нем закрытое поле типа bool IsLoaded
+2. Создать private-метод Reload, который будет проверять, заряжено наше оружие и выводить соответствующую запись на консоль.
+3. Создать public-метод Shoot, который будет будет проверять, заряжено ли наше оружие. Если нет, то заряжать его.
