Задание 3
Создайте проект по шаблону "Console Application". Создайте метод «private static int[]
SortArray(bool isAscending, params int[] array)». Метод должен отсортировать массив и вернуть
отсортированный массив в виде результата. Если параметр isAscending равен true - сортировать
по возрастанию, если false - сортировать по убыванию. Как организовать алгоритм сортировки,
полностью зависит от вашего выбора. Вызвать метод SortArray в контексте задачи для большого
массива типа int. Результат задачи обработать в продолжении, где нужно вывести на экран
консоли все элементы массива через запятую.

Решение:
-1. Создать метод SortArray(bool isAscending, params int[] array), использовать необязательный параметр установить bool IsAscendind = true;
-2. В методе с помощью цикла for произвести сравнение на больше меньше каждого элемента массива;
-3. Использовать флаг TaskCreationOptions.LongRunning, так как метод может принимать переменное количество аргументов и количество аргументов сложно предугадать;
-4. В Main() запустить метод в контексте задачи Task передав в конструктор метод и аргументы;
-5. На экземпляре класса Task вызвать метод ContinueWith() и с помощью лямбда-выражения в цикле for вывести отсортированный массик на экран.
-6. Подождать выполнение метода вызвав на экземпляре задачи метод Wait();
